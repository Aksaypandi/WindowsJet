<strong><span class="dropcap">E</span>nable Work Folders Client in Windows 10: </strong>The files of the Work Folders Client are saved on the Sync Share server and hence make it useful to access the files from anywhere and from any devices. This article will illustrate the silhouette of <strong>how to enable Work Folders Client in windows 10.</strong>

<ul>
 	<li><a href="#1"><strong>Enable Work Folders Client in Windows 10</strong></a></li>

</ul>
<h2 id="1">Clear cut ways to Enable Work Folders Client in Windows 10:</h2>
Follow the below steps to<strong> Enable Work Folders Client in Windows 10.</strong>
<ul>
 	<li>Go to the <strong>Start menu</strong> and search for <strong>Control Panel</strong> and open it.</li>
</ul>
[caption id="attachment_378" align="aligncenter" width="488"]<img class="wp-image-378 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_5-2.png" alt="Control Panel" width="488" height="778" /> Control Panel[/caption]
<ul>
 	<li>In the <strong>Control Panel window</strong>, you have to click on <strong>Programs</strong>.</li>
</ul>
[caption id="attachment_379" align="aligncenter" width="1141"]<img class="wp-image-379 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_6-2.png" alt="Programs" width="1141" height="737" /> Programs[/caption]
<ul>
 	<li>Then, select the <strong>Turn Windows features on or off </strong>in the Programs dialog box.</li>
</ul>
[caption id="attachment_380" align="aligncenter" width="1163"]<img class="wp-image-380 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_7-2.png" alt="Programs Windows" width="1163" height="730" /> Programs Windows[/caption]
<ul>
 	<li>The <strong>Windows Features dialog box</strong> will appear, From the list of the features available in that, scroll down and locate Windows <strong>Work Folders Client</strong> and enable it, then click <strong>OK</strong>.</li>
</ul>
[caption id="attachment_734" align="aligncenter" width="844"]<img class="wp-image-734 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_16.png" alt="Windows Features" width="844" height="589" /> Windows Features[/caption]
<ul>
 	<li>Now, Windows will search for the required files.</li>
</ul>
[caption id="attachment_381" align="aligncenter" width="791"]<img class="wp-image-381 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_1-1.png" alt="Searching Files" width="791" height="682" /> Searching Files[/caption]
<ul>
 	<li>Then it apply the necessary changes.</li>
</ul>
[caption id="attachment_640" align="aligncenter" width="1067"]<img class="wp-image-640 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_14.png" alt="Apply Changes" width="1067" height="693" /> Apply Changes[/caption]
<ul>
 	<li>Finally, <strong>Windows completed the required changes</strong> and you can click <strong>Close</strong>.</li>
</ul>
[caption id="attachment_384" align="aligncenter" width="1010"]<img class="wp-image-384 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_4-1.png" alt="Requested changes" width="1010" height="677" /> Requested changes[/caption]
<ul>
 	<li><strong>Windows Work Folders Client</strong> is now enabled on Windows 10.</li>
</ul>
