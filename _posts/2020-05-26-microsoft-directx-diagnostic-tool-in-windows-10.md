This article will illustrate the silhouette of <strong>how to Open and Use Microsoft DirectX Diagnostic Tool in Windows 10.</strong>

<h2 id="1">Clear cut ways to open Microsoft DirectX Diagnostic Tool in Windows 10:</h2>
The different ways to open <strong>Microsoft DirectX Diagnostic Tool in Windows 10 </strong>simply, follow the below steps.
<h3><strong>Way 1: Open Microsoft DirectX Diagnostic Tool using the Run command</strong></h3>
<ul>
 	<li>Press <strong>Windows+R </strong>to open the<strong> Run dialog box.</strong></li>
 	<li>In the <strong>Run dialog box</strong>, type as <strong>dxdiag </strong>in the <strong>open bar</strong> and click<strong> OK.</strong></li>
</ul>
[caption id="attachment_2336" align="aligncenter" width="498"]<img class="wp-image-2336 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_1-30.png" alt="Run Command Dialog Box" width="498" height="334" /> Run Command Dialog Box[/caption]
<ul>
 	<li>Now, the <strong>Microsoft DirectX Diagnostic Tool </strong>will be opened on your screen.</li>
</ul>
[caption id="attachment_2337" align="aligncenter" width="1063"]<img class="wp-image-2337 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_2-27.png" alt="DirectX Tool" width="1063" height="695" /> DirectX Tool[/caption]
<h3><strong>Way 2: Open </strong><strong>Microsoft DirectX Diagnostic Tool </strong><strong>using PowerShell</strong></h3>
<ul>
 	<li>Go to<strong> start menu →</strong> In the<strong> search bar →</strong> Type as <strong>PowerShell.</strong></li>
 	<li>Then the<strong> PowerShell icon</strong> is visible on the top of the <strong>start menu.</strong></li>
 	<li>Click the icon the<strong> PowerShell window</strong> gets open.</li>
</ul>
[caption id="attachment_2338" align="aligncenter" width="534"]<img class="wp-image-2338 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_3-27.png" alt="Powershell" width="534" height="797" /> Powershell[/caption]
<ul>
 	<li>In <strong>PowerShell window,</strong> type as <strong>dxdiag </strong>and press<strong> Enter.</strong></li>
</ul>
[caption id="attachment_2339" align="aligncenter" width="1157"]<img class="wp-image-2339 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_4-10.png" alt="Powershell" width="1157" height="535" /> Powershell[/caption]
<ul>
 	<li>The <strong>Microsoft DirectX Diagnostic Tool </strong>will open on your screen.</li>
</ul>
<h3><strong>Way 3: Create a shortcut to open Microsoft DirectX Diagnostic Tool</strong></h3>
<ul>
 	<li>To create a<strong> shortcut</strong> for <strong>Microsoft DirectX Diagnostic Tool</strong>,<strong> Right-click</strong> on your desktop, the<strong> popup menu</strong> gets open.</li>
 	<li>In that select<strong> New</strong> option <strong>→</strong> choose<strong> Shortcut.</strong></li>
</ul>
[caption id="attachment_2340" align="aligncenter" width="974"]<img class="wp-image-2340 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_5-5.png" alt="Popup menu" width="974" height="639" /> Popup menu[/caption]
<ul>
 	<li>In the <strong>Create Shortcut</strong> window, you have to type <strong>dxdiag</strong><b> </b>in the given space of <strong>Type the location of the item</strong>. and click <strong>Next</strong> to proceed.</li>
</ul>
<p style="background: aquamarine;"><code>dxdiag</code></p>


[caption id="attachment_2341" align="aligncenter" width="957"]<img class="wp-image-2341 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_6-3.png" alt="Location Window" width="957" height="623" /> Location Window[/caption]
<ul>
 	<li>Then in the next window, give the name as you want for the shortcut or leave the default name and click <strong>Finish</strong>.</li>
</ul>
[caption id="attachment_2342" align="aligncenter" width="973"]<img class="wp-image-2342 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_7-3.png" alt="Name Window" width="973" height="619" /> Name Window[/caption]
<ul>
 	<li>The <strong>shortcut for Microsoft DirectX Diagnostic Tool </strong>has been created on your <strong>desktop.</strong></li>
</ul>
[caption id="attachment_2343" align="aligncenter" width="1197"]<img class="wp-image-2343 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_8-2.png" alt="Shortcut" width="1197" height="700" /> Shortcut[/caption]
<ul>
 	<li><strong>Double-click</strong> on the icon to open the<strong> Microsoft DirectX Diagnostic Tool </strong>or Else <strong>right-click</strong> on the icon and select open, the <strong>Microsoft DirectX Diagnostic Tool </strong>gets open.</li>
</ul>
<h2 id="2">Get started with Microsoft DirectX Diagnostic Tool in Windows 10:</h2>
The diagnostic tool contains the following options. Here we have discussed brief information about Microsoft DirectX Diagnostic Tool in Windows 10.
<ul>
 	<li><strong>System</strong></li>
 	<li><strong>Display</strong></li>
 	<li><strong>Render</strong></li>
 	<li><strong>Sound</strong></li>
 	<li><strong>Input</strong></li>
</ul>
<h3>System:</h3>
<ul>
 	<li>This option gives you a lot of information related to your system.</li>
 	<li>Refer to the below screenshot to know the information about your system.</li>
</ul>
[caption id="attachment_2344" align="aligncenter" width="1073"]<img class="wp-image-2344 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_9-2.png" alt="System" width="1073" height="705" /> System[/caption]
<h4><strong>To Check for WHQL Digital Signatures:</strong></h4>
<ul>
 	<li>The WHQL stands for Windows Hardware Quality Labs.</li>
 	<li>Check in the<strong> WHQL Digital Signature, </strong>which is mentioned in the below screenshot.</li>
 	<li>It will verify driver packages for <strong>Windows Hardware Certification Kit(HCK)</strong>.</li>
</ul>
[caption id="attachment_2345" align="aligncenter" width="1074"]<img class="wp-image-2345 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_10-2.png" alt="System" width="1074" height="716" /> System[/caption]
<ul>
 	<li>Then you can <strong>save all the information</strong> as a text file by clicking Save All Information.</li>
</ul>
[caption id="attachment_2346" align="aligncenter" width="1125"]<img class="wp-image-2346 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_11-2.png" alt="System" width="1125" height="730" /> System[/caption]
<h3>Display:</h3>
<ul>
 	<li>To go to this option, simply click the <strong>Next page option or click the display option</strong> which appears at the top.</li>
 	<li>Here, you can find a piece of information about the <strong>Device </strong>and <strong>Drivers</strong> and <strong>DirectX Features</strong>.</li>
 	<li>In addition, you can also find that these drivers and hardware are in proper functioning, once it checks if there is no problem, then you can find <strong>No problems found in the </strong>Notes section.</li>
</ul>
[caption id="attachment_2347" align="aligncenter" width="1098"]<img class="wp-image-2347 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_12-1.png" alt="Display" width="1098" height="699" /> Display[/caption]
<h3>Render:</h3>
<ul>
 	<li>Click the <strong>render option or next page</strong> to go for this option.</li>
 	<li>You can find all the information related to the <strong>Graphics Card</strong>.</li>
 	<li>Here, you can check for the normal functioning of the Graphics card.</li>
 	<li>Refer to the <strong>below screenshot</strong> to know the information includes in this option.</li>
</ul>
[caption id="attachment_2348" align="aligncenter" width="1081"]<img class="wp-image-2348 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_13-1.png" alt="Render" width="1081" height="722" /> Render[/caption]
<h3>Sound:</h3>
<ul>
 	<li>This tool gives you a piece of information about your Device Speakers such as <strong>Hardware ID</strong>, <strong>Manufacturer ID</strong>, &amp; more.</li>
</ul>
[caption id="attachment_2349" align="aligncenter" width="1106"]<img class="wp-image-2349 size-full" src="https://windowsjet.com/wp-content/uploads/2020/05/Screenshot_14-1.png" alt="Sound" width="1106" height="715" /> Sound[/caption]
<h3>Input:</h3>
<ul>
 	<li>In this tool, you will find information on<strong> DirectInput devices.</strong></li>
 	<li>Finally, click on <strong>Exit</strong> to quit the <strong>DirectX Diagnostic Tool in Windows 10.</strong></li>
</ul>
