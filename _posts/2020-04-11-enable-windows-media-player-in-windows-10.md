<strong><span class="dropcap dropcap1">E</span></strong><strong>nable Windows Media Player in Windows 10: Windows Media Player </strong>is used to <strong>play music, video files</strong> and also to<strong> view photos </strong>in personal computers. Enabling this media player in your device is quite easy and simple. Let's follow the guidelines given below in order to <strong>Enable Windows Media Player in Windows 10</strong>.
<h2>Jump To:</h2>
<ul>
 	<li><a href="#1"><strong>Guidelines to Enable Windows Media Player</strong></a></li>
 	<li><a href="#2"><strong>Conclusion</strong></a></li>
</ul>
<h2 id="1">Guidelines to Enable Windows Media Player:</h2>
<ul>
 	<li>Click the <strong>Start</strong> button, type <b>Control Panel </b>and click<b> Enter</b>.</li>
</ul>
[caption id="attachment_481" align="alignnone" width="900"]<img class="size-full wp-image-481" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_1-7.png" alt="Control Panel" width="900" height="500" /> Control Panel[/caption]
<ul>
 	<li>In the <strong>Control Panel window</strong>, you have to click on <strong>Programs</strong>.</li>
</ul>
[caption id="attachment_482" align="alignnone" width="900"]<img class="size-full wp-image-482" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_2-7.png" alt="Programs" width="900" height="500" /> Programs[/caption]
<ul>
 	<li>Now, you have to select <strong>Turn Windows features on or off</strong>.</li>
</ul>
[caption id="attachment_483" align="alignnone" width="900"]<img class="size-full wp-image-483" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_3-7.png" alt="Turn Windows features on or off" width="900" height="500" /> Turn Windows features on or off[/caption]
<ul>
 	<li>A pop-up window will appear. Scroll down and expand <strong>Media Features.</strong></li>
 	<li>You have to select the check-box of the<strong> Windows Media Player</strong> and click <strong>OK</strong>.</li>
</ul>
<img class="alignnone size-full wp-image-484" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_4-7.png" alt="" width="900" height="500" />
<ul>
 	<li>Now, Windows will look for the <strong>required files</strong>.</li>
</ul>
[caption id="attachment_485" align="alignnone" width="900"]<img class="size-full wp-image-485" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_5-5.png" alt="Required files" width="900" height="550" /> Required files[/caption]
<ul>
 	<li>It will <strong>apply the changes</strong> it made.</li>
</ul>
[caption id="attachment_486" align="alignnone" width="898"]<img class="size-full wp-image-486" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_6-4.png" alt="Applying changes" width="898" height="525" /> Applying changes[/caption]
<ul>
 	<li>Once the Windows completed the necessary changes, you have to click <strong>Close</strong>.</li>
</ul>
[caption id="attachment_487" align="alignnone" width="900"]<img class="size-full wp-image-487" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_7-4.png" alt="Close" width="900" height="550" /> Close[/caption]
<ul>
 	<li>Windows Media Player should now be enabled in your device.</li>
</ul>
