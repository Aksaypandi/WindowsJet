<h2 id="1">Easy steps to create a shortcut for MS Paint in Windows 10:</h2>
In order to create a shortcut for <strong>MS Paint in Windows 10</strong> then you need to follow the below steps. Once you create the shortcut you don't have to depend on <strong>Run command</strong> or <strong>Start menu.</strong> Just <strong>Double-clicking</strong> the icon you can start using the <strong>shortcut</strong> for any instance <strong>easily and quickly.</strong>
<ul>
 	<li>Simply, <strong>Right-click</strong> on your desktop and select <strong>New.</strong></li>
 	<li>In the <strong>Popup menu</strong> choose the <strong>Shortcut.</strong></li>
</ul>
[caption id="attachment_1003" align="aligncenter" width="1002"]<img class="wp-image-1003 size-full" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_1-26.png" alt="Popup menu" width="1002" height="676" /> Popup menu[/caption]
<ul>
 	<li>The <strong>create shortcut window</strong> will appear in that, you have to enter <b><strong>mspaint</strong> </b>in <strong>Type the Location of the item box.</strong> Click <strong>Next</strong> to continue.</li>
</ul>
<p style="background: aquamarine;"><code>mspaint</code></p>

<ul>
 	<li>Now, you can give any name for this shortcut as per your wish in the given space box of <strong>Type a name for this shortcut </strong>or else leave the default name as it is.</li>
 	<li>Then, click <strong>Finish</strong>.</li>
</ul>
