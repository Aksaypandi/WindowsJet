<strong><span class="dropcap dropcap1">D</span></strong><strong>eveloper Settings Shortcut in Windows 10: Developer Settings in Windows 10</strong> is mainly <strong>intended for developer purposes</strong> only. Developers can make use of this feature to enable the <strong>device portal, device discovery or to manage file explorer settings</strong>. Let's see quick instructions to create a <strong>Shortcut for Developer Settings in Windows 10</strong>.

<h2 id="1">Quick Instructions to Create a Shortcut for Developer Settings:</h2>
Follow the below steps to create <strong>Developer Settings </strong><strong>Shortcut in Windows 10</strong>. After that, start to access the settings instantly with the use of this shortcut.
<ul>
 	<li>You just need to <strong>Right-click</strong> on the <strong>Desktop</strong> and select <strong>New</strong>. After that, choose <strong>Shortcut </strong>from the pop-up menu.</li>
</ul>
[caption id="attachment_704" align="alignnone" width="900"]<img class="size-full wp-image-704" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_1-19.png" alt="Developer Settings - Shortcut Creation" width="900" height="500" /> Developer Settings - Shortcut Creation[/caption]
<ul>
 	<li>You have to enter <strong>ms-settings:developers </strong>in the text box where you have asked to <strong>Type the location of the item </strong>and then click <strong>Next</strong>.</li>
</ul>
<p style="background: Tomato;"><code style="background: Tomato; color: white;">ms-settings:developers</code></p>


[caption id="attachment_705" align="alignnone" width="900"]<img class="size-full wp-image-705" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_1-20.png" alt="Developer Settings - Shortcut Location" width="900" height="519" /> Developer Settings - Shortcut Location[/caption]
<ul>
 	<li>In the subsequent window, type the name as <strong>Developer Settings</strong> in the given space for <strong>Type a name for this shortcut.</strong></li>
 	<li>Then, click <strong>Finish</strong>.</li>
</ul>
[caption id="attachment_706" align="alignnone" width="900"]<img class="size-full wp-image-706" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_2-19.png" alt="Developer Settings - Shortcut Name" width="900" height="520" /> Developer Settings - Shortcut Name[/caption]
<ul>
 	<li>Ultimately, the <strong>Developer Settings</strong><b> shortcut</b> is now created on your desktop.</li>
</ul>
[caption id="attachment_707" align="alignnone" width="900"]<img class="size-full wp-image-707" src="https://windowsjet.com/wp-content/uploads/2020/04/Screenshot_3-16.png" alt="Developer Settings - Shortcut Created" width="900" height="520" /> Developer Settings - Shortcut Created[/caption]
<ul>
 	<li>You can access this tool with the use of this shortcut icon in any instance.</li>
</ul>
